# birthday_app

Currently supported calendars:
- Julian
- Hebrew
- Islamic (Hijri)
- Persian
- Mayan
- Indian Civil Calendar
- French Republican Calendar

## Project setup
```
npm install
npm run serve
npm run build
npm run deploy
```
